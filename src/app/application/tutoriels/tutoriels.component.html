C'est quoi Angular?

Angular est un frameworkpour développement web gratuit (Open Source) SPA (Single Page Application) en Typescript. Il permet au développeurs de coder plus rapidement grâce aux commandes angulars qui permet de générer des templates.

Angular est un framework Open Source pour le developpement front-end rapide de pages web SPA à l'aide du Typescript.

Un site dévelopé avec Angular est adapté pour toute platforme (Ordinateur, Mobile, etc.) et Responsive

De plus, Angular est un des frameworks Javascript les plus populaire principalement maintenu par google.

Préparation de votre environnement pour Angular 10

Npm?

C'est quoi npm? Npm (node package manager) est le gestionnaire de "package" javascript et permet de placer les modules dans leur répertoire respective.

Tout d'abord il faut télécharger et installer node.js
Voici le lien pour téléchargé Node.js

IMPORTANT!!! Si vous avez déjà un terminal d'installé qui permet d'exécuté des commandes linux, alors passer à la prochaine étape.

Suite au téléchargement et à l'installation de Node.js, il faut télécharger GitBash.
Télécharger selon votre système d'exploitation

Une fois que vous avez Node.js d'installé, ainsi qu'un terminal disponible, exécutez les commandes suivantes pour vérifié que Node.js fonctionne:
node -v
ensuite
npm -v

Si les commandes retourne une erreur, alors redémarrer votre ordinateur et essayer à nouveau.

**Si ça ne fonctionne toujours pas veuillez consulter l'article suivant:
https://love2dev.com/blog/node-is-not-recognized-as-an-internal-or-external-command/

Installation de Angular 10 (Suite... )

Une fois que tout fonctionne, exécuté les commandes suivantes dans le répertoire de votre choix pour vos projets angular
Pour se pointer dans le répertoire de votre choix utilisez la syntaxe comme suit:
cd répertoire/chemin ou cd .. pour remonter d'un niveau.

npm install -g @angular/cli (pour installer angular dans le répertoire de manière globallement)
npm install -g typescript --save-dev(pour installer typescript dans le répertoire de manière globallement)
pour plus d'information sur Typescript visitez: https://www.typescriptlang.org/download
ng new my-first-project (syntax pour créer un nouveau projet)
cd my-first-project (commande pour se pointer dans le projet Angular)
npm install nomDeLaLibrairie(commande pour installer une librairie externe à l'aide de NPM)
