About this project

Reason for this project

The reason behind this project can be split into two. The first is for acedemic purposes, since this is a graded assignement. The second reason for this project is to test all the knowledge I've gained throughout the entire semester. This semester the main topics were Bootstrap, Angular and Typescript.

Bootstrap

In Bootstrap we learned to use Containers, adapt Bootstrap Templates, make the website Responsive and code faster with Bootstrap styles.

Angular

In Angular we learned to about Components, the angular files, the commands and the power of angular to create SPA websites like this one

Typescript
Typescript was used for writing Get, Post, Put, Delete methods as well as other methods for validation.

Goal of this project

My personal goals: The goal of this project is to test my skills with the deadline given and spread the love for Hololive. In this project i'm expected to make use of Containers, Margins, Table, Card, Footer, Navbar, Forms, and the Responsiveness of the website

  • Create a responsive website
  • Make the website look good to the eyes
  • Contribute to sharing Hololive to others
  • Use modern web design techniques

Future of this project

This project is by no means perfect and I expect to bring modifications to it. In the future the website will be more modern as I learn more techniques and ways to design it.
